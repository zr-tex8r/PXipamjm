PXipamjm パッケージバンドル
===========================

LaTeX: アレ

### 前提環境

  * フォーマット： LaTeX
  * エンジン： pTeX / upTeX
  * DVIウェア： dvipdfmx (たぶん)
  * 依存パッケージ/ソフトウェア：
      - ipamjm パッケージ
      - IPAmj明朝フォント
        (ipamjm と合致する版)

### インストール

  - `*.sty` → $TEXMF/tex/latex/PXipamjm
  - `*.tfm` → $TEXMF/fonts/tfm/public/PXipamjm
